# Genesis - Workout Dashboard

A professional, feature-rich 8-week workout transformation tracker built with Vanilla JS, CSS, and HTML.

## 🚀 Features

- **Progress Visualization:** Track your 8-week journey with an interactive dashboard and real-time progress rings.
- **Streak Tracking:** Stay motivated with a streak counter that tracks consecutive workout days.
- **Detailed Notes:** Log sets, reps, and how you felt for every single workout session.
- **Dark & Light Mode:** Seamlessly switch between themes based on your preference.
- **CSV Export:** Download your entire progress history as a CSV file for long-term tracking.
- **Mobile Responsive:** Designed to work perfectly on both desktop and mobile devices.
- **Local Persistence:** Your data is saved locally in your browser, no login required.

## 🏋️ Workout Plan

### Weekly Split

#### Monday — Workout A (Full Body / Deadlift Focus)
- **Deadlift** — 3×8
- **Overhead Press** — 3×8
- **Bench Press** — 3×8
- **Bicep Curl** — 3×10
- **Overhead Triceps Extension** — 3×10
- **Calf Raises** — 3×20
- **Sit-Ups** — 3×20

#### Tuesday — Cardio + Core
- Brisk walk / jog / cycle — 30 min
- **Plank** — 3×30–45 sec
- **Leg Raises** — 3×15
- **Mountain Climbers** — 3×20

#### Wednesday — Workout B (Squat Focus)
- **Back Squat** — 3×8
- **Bent-Over Row** — 3×8
- **Incline Press** — 3×8
- **EZ Bar Curl** — 3×10
- **Lateral Raise** — 3×10
- **Tricep Pushdown** — 3×10
- **Oblique Cable Twist** — 3×20

#### Thursday — Active Recovery & Mobility
- Easy walk / Yoga / Swim — 20–30 min
- **Hip Flexor Stretch**
- **Chest Opener Stretch**
- **Thoracic Rotation**

#### Friday — Workout C (Compound Focus)
- **Split Squat** — 3×8
- **Close-Grip Bench** — 3×8
- **Flat Dumbbell Press** — 3×8
- **Incline Dumbbell Curl** — 3×10
- **Lat Pulldown** — 3×10
- **Face Pull** — 3×10
- **Hanging Leg Raise** — 3×20

#### Saturday — Cardio Day
- Jog / Cycle / Elliptical — 30–40 min
- **Jump Rope** — 3×2 min
- Intervals: 1 min fast / 2 min easy × 8–10 rounds

#### Sunday — Full Rest
- Sleep, Recover, Eat enough protein.

## 🧐 Critique & Optimization
- **Deadlift 3x8** can be fatiguing; consider **3x5** for strength focus.
- Add **pull-ups/assisted pull-ups** eventually for back width.
- Use **face pulls** to balance heavy pressing days and protect shoulders.
- **Weekly Progression:**
  - Week 1: Easy/Baseline
  - Week 2: +2.5–5 kg on compounds
  - Week 3: +2.5–5 kg
  - Week 4: Deload (reduce volume/intensity)

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Modern layout with Flexbox, Grid, and CSS Variables.
- **JavaScript (ES6+):** Pure Vanilla JS for state management and interactivity.

## 📖 How to Use

1. Open `index.html` in any modern web browser.
2. Mark workouts as completed by clicking on the day cards.
3. Click the **Note** button to add details for any specific day.
4. Watch tutorials by clicking the **Video** button or "Watch Video" in details.
5. Export your data anytime using the "Export CSV" button.
