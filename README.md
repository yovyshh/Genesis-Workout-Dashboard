# Genesis - 12-Week Transformation Dashboard

A professional, feature-rich 12-week (3-month) workout transformation tracker focused on Calisthenics and Dumbbell exercises. This dashboard helps you stay consistent, track progress, and master advanced bodyweight skills.

## 🚀 Features

- **12-Week Roadmap:** Three distinct phases (Foundation, Intensity, Advanced) to keep you progressing.
- **Progress Visualization:** Track your 84-day journey with an interactive dashboard and real-time progress rings.
- **Performance Analytics:** View consistency charts and focus distribution.
- **Month Tabs:** Quickly jump between training phases (Month 1, 2, and 3).
- **Tutorial Videos:** Integrated YouTube links for every single exercise to ensure perfect form.
- **Detailed Notes:** Log sets, reps, and energy levels for every session.
- **Theme Support:** Modern dark theme with easy navigation.
- **Mobile Optimized:** Persistent bottom navigation for a native app feel on mobile devices.

## 🏋️ Workout Plan

### Phase 1: Foundation (Month 1)
- **Schedule:** 3 days per week (Full Body).
- **Focus:** Building the neural base and perfecting form.
- **Key Moves:** Push-ups, Dumbbell Floor Press, Pull-ups, Goblet Squats.

### Phase 2: Intensity (Month 2)
- **Schedule:** 4 days per week (Upper/Lower Split).
- **Focus:** Increasing volume and adding dumbbell load.
- **Key Moves:** Pike Push-ups, DB Shoulder Press, RDLs, Glute Bridges.

### Phase 3: Advanced (Month 3)
- **Schedule:** 4–5 days per week (Advanced Split).
- **Focus:** Mastering elite skills and maximizing hypertrophy.
- **Key Moves:** Archer Push-ups, Pistol Squats, Muscle-up Progressions, Windshield Wipers.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Flex/Grid), Vanilla JavaScript (ES6+).
- **Storage:** LocalStorage for persistence.
- **Assets:** Google Fonts (Inter, JetBrains Mono), Flaticon.

## 📖 How to Use

1. Open `index.html` in your browser.
2. Use the **Month Tabs** to navigate between phases.
3. Click on a day card to see the workout details and tutorial links.
4. Mark workouts as completed and save notes to track your strength gains.
5. Export your data via **CSV** anytime for external logging.
