# Elite Performance - Workout Dashboard

A professional, feature-rich 4-week workout transformation tracker built with Vanilla JS, CSS, and HTML.

## 🚀 Features

- **Progress Visualization:** Track your 4-week journey with an interactive dashboard and real-time progress rings.
- **Streak Tracking:** Stay motivated with a streak counter that tracks consecutive workout days.
- **Detailed Notes:** Log sets, reps, and how you felt for every single workout session.
- **Dark & Light Mode:** Seamlessly switch between themes based on your preference.
- **CSV Export:** Download your entire progress history as a CSV file for long-term tracking.
- **Mobile Responsive:** Designed to work perfectly on both desktop and mobile devices.
- **Local Persistence:** Your data is saved locally in your browser, no login required.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Modern layout with Flexbox, Grid, and CSS Variables.
- **JavaScript (ES6+):** Pure Vanilla JS for state management and interactivity.

## 📖 How to Use

1. Open `index.html` in any modern web browser.
2. Mark workouts as completed by clicking on the day cards.
3. Click the 📝 icon to add notes to any specific day.
4. Watch tutorials by clicking the ▶️ icon (where available).
5. Use the sidebar to switch between themes or view your streak.
6. Export your data anytime using the "Export CSV" button.

## 📂 Project Structure

- `index.html`: Main application entry point.
- `style.css`: Modern design system and layouts.
- `script.js`: Application logic and state management.
